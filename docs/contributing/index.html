Contributing

Newcomers are always welcome to contribute to gearbox, and we are happy to offer help to newcomers. Before making changes, please first discuss the change you want to make through Discord, or email us at gearbox@googlegroups.com

There are many ways to contribute:#

  • Suggest new features to be implemented
  • Report issues
  • Fixing issues
  • Improve Documentation

Code Contribution#

If you wish to work on an issue, please comment on the issue that you want to work on it. This is to prevent duplicated efforts on the same issue.
